- khởi tạo repository: git init

- púsh code từ local lên remote: 3 bước
+ chọn: git add -A ( hit status : show thông tin)
+ xác nhận: git commit -m "nội dung commit"
+ đẩy: lần đầu ~ git push -u origin master
       lần 2 trở lên ~ git push


* Branch
 - tạo nhánh: git checkout -b login
 - chuyển nhánh: git checkout master
